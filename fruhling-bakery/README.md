How to run:
1. On your machine, copy this folder into a project directory or use it as-is.
2. Run:
   npm install
   npm start

Notes:
- Styling is preserved from the original HTML in src/App.css (Option A).
- Images are loaded from the S3 bucket used in the original site.
- The original uploaded HTML is included as ronis-website.html for reference.
